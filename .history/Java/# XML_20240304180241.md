# XML

XML格式能够表达结构层次，并且允许在确保格式不走样的情况下记录重复元素。

```XML
<congfig>
    <entry id = "title">
        <font>
            <name>Helvetica</name>
            <size>36</size>
        <font>
    </entry>
    <entry id = "body">
        <font>
            <name>Times Roman</name>
            <size>16</size>
        </font>
    </entry>
```
