## 第二部分：后端服务分解

### 2. 数据查询服务

#### **2.1 mmrels01_inq - 头部指标查询**

**文件位置**：`server/Dev/MM/MMRE/p_mmre_18540/mmrels01_inq.cpp`

**功能**：查询当日采购、销售、厂内倒运数量统计

**SQL查询**：

```sql
SELECT  
  SUM(CASE WHEN BUSY_TYPE IN('采购进厂') THEN 1 ELSE 0 END) AS CG_COUNT,  
  SUM(CASE WHEN BUSY_TYPE IN('销售') THEN 1 ELSE 0 END) AS XS_COUNT,  
  SUM(CASE WHEN BUSY_TYPE NOT IN('销售', '采购进厂') THEN 1 ELSE 0 END) AS CN_COUNT  
  FROM TYMREC1  
  WHERE WORK_DATE = @WORK_DATE
```

**数据表**：TYMREC1（资源作业履历表）

#### **2.2 mmrels02_inq - 车运作业详情查询**

**文件位置**：`server/Dev/MM/MMRE/p_mmre_18540/mmrels02_inq.cpp`

**功能**：查询当日车运作业详细信息

**SQL查询**：

```sql
SELECT  ROW_NUMBER() OVER (ORDER BY T1.WORK_SEQ_NO) AS ROW_NUM,  
  T1.TRUCK_NO || ' ' || T1.CAR_NO AS TRUCK_NO  
  , T2.MAT_CNAME 
  , T3.STOCK_CNAME AS SRC_STOCK_CNAME 
  , T4.STOCK_CNAME AS DST_STOCK_CNAME 
  , T1.BASE_RESOURCE 
  FROM tymrec1 T1 
  LEFT JOIN TYMRE21 T2 ON T1.MAT_CODE = T2.MAT_CODE 
  LEFT JOIN TYMRE11 T3 ON T3.STOCK_CODE = T1.SRC_STOCK_CODE 
  LEFT JOIN TYMRE11 T4 ON T4.STOCK_CODE = T1.DST_STOCK_CODE 
  WHERE WORK_DATE = @WORK_DATE  
  ORDER BY T1.WORK_SEQ_NO
```

**关联表**：

- TYMREC1：资源作业履历表
- TYMRE21：物料信息表
- TYMRE11：库区信息表

#### **2.3 mmrels03_inq - 月度统计数据查询**

**文件位置**：`server/Dev/MM/MMRE/p_mmre_18520/mmrels03_inq.cpp`

**功能**：查询月度采购、销售、厂内倒运重量统计

**SQL查询**：

```sql
SELECT  
 SUM(CASE WHEN BUSY_TYPE IN('采购进厂', '进厂直供', '领用') THEN BALANCE_WGT ELSE 0 END) AS MON_CG, 
 SUM(CASE WHEN BUSY_TYPE IN('销售') THEN BALANCE_WGT ELSE 0 END) AS MON_XS, 
 SUM(CASE WHEN BUSY_TYPE NOT IN('销售', '采购进厂', '进厂直供', '领用') THEN BALANCE_WGT ELSE 0 END) AS MON_CN 
 FROM TYMREC1 
 WHERE WORK_DATE BETWEEN SUBSTR(@WORK_DATE,1,6) || '01' AND @WORK_DATE
```

**数据表**：TYMREC1（资源作业履历表）
