# Microsoft Activation Scripts (MAS)使用手册

1. 打开Windows菜单找到PowerShell并打开 
   ![图片](/闲鱼/MAS/img/Powershell.png "Powershell")
2. 输入 irm https://gitee.com/cmontage/mas-cn/raw/main/GETMASCN.ps1 | iex
   ![图片](/闲鱼/MAS/img/指令.png "指令")
3. 会弹出一个链接，点击后等待一会儿会自动跳转出cmd窗口然后根据指示操作
4. 在cmd中输入2，使用Ohook激活
    ![图片](/闲鱼/MAS/img/激活界面.png)
5. 跳转到安装界面输入1
    ![图片](/闲鱼/MAS/img/安装界面.png)
6. 激活完成
    ![图片](/闲鱼/MAS/img/完成页面.png)
7. 可以愉快使用Office啦！
